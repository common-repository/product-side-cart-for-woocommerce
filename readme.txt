== Description ==

<Strong>Product Side Cart For Woocommerce</Strong> is help customers fulfill their orders with a single click. The plugin also provides users with many useful tools to design all the elements relating such as Sidebar Cart, Menu Cart, Sticky Add To Cart button, Checkout on Sidebar Cart, which will make your WooCommerce Store to work more effectively.

<a href="https://topsmodule.com/how-to-create-product-side-cart-for-woocommerce/"><strong>Documentation</strong></a>
<a href="https://topsmodule.com/task/prodcut-side-cart-for-woocommerce/"><strong>Live Demo</strong></a>

<h3>THE PLUGIN FEATURES</h3>

<ul>
<li>Display a sidebar cart icon(Enable/ Disable)</li>
<li>Hide/ Show cart basket on desktop and mobile view</li>
<li>Hide/ Show product count</li>
<li>Enable/ disable fly to cart animation</li>
<li>Display Product count as number of product or sum of quantity of all product</li>
<li>Hide/ Show cart basket on post type, pages</li>
<li>Hide/ Show close icon</li>
<li>Side cart body setting</li>
<li>Enable/ disable side cart product slider</li>
<li>Enable/ disable subtotal, shipping total and main total</li>
<li>Customize side cart width</li>
<li>Change heading and close icon size</li>
<li>Set heading and close icon position</li>
<li>Customize header background color, border color/style and title color</li>
<li>Customize cart body with background color, product image width, border radius</li>
<li>Customize slider background, button background and text color</li>
<li>Customize product slider image width and font size</li>
<li>Hide/ Show product slider image, title, price and add to cart button</li>
<li>Change footer background color</li>
<li>Set footer button position with padding</li>
<li>Set footer button row</li>
<li>Customize cart, checkout and continue shopping button background color, button hover color, text Color.</li>
<li>Set backet position or shape</li>
<li>Select basket icon with basket background color and basket color</li>
<li>Add basket icon size</li>
<li>Set basket position</li>
<li>Add custom basket image</li>
<li>Select basket count position with background color and text color</li>
<li>Select cart order list in asc and desc</li>
<li>Change delete and close icon</li>
<li>Set product delete as a icon or text with color</li>
<li>Customize delete and close icon with icon color and icon hover color</li>
<li>Remove Product From Cart.</li>
<li>Display Shipping Amount.</li>
<li>Update quantity of each product.</li>
<li>Text/ Url setting</li>
</ul>

== Installation ==
1. Make sure you have woocommerce installed.
2. Download the file to install and activate.
3. Done! Just remember to keep the plugins up to date!
